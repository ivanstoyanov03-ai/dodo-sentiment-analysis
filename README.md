## 🎯 Цель проекта

Разработка сквозного аналитического конвейера для сбора, очистки, визуализации и бинарной классификации тональности отзывов клиентов сети пиццерий «Додо Пицца» с применением методов NLP.

---

## 🔬 Ключевые задачи

- Парсинг и агрегация отзывов с платформ (Яндекс Карты, Otzovik, IRecommend)
- Многоуровневая предобработка кириллического текста (лемматизация, стоп-слова)
- Разведочный анализ с визуализацией частотных маркеров (WordCloud)
- Обучение модели бинарной классификации (TF‑IDF + логистическая регрессия)
- Создание интерактивного дашборда для мониторинга лояльности

---

## 📊 Результаты

- **Accuracy на тестовой выборке:** 86%
- **Ключевые драйверы негатива:** логистические сбои («опоздал курьер», «холодная пицца»)
- **Интерактивный дашборд** (Plotly) — консолидация метрик по платформам
